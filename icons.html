Grids


Purpose

Icons serve as a visual representation for a tool or function.


Button and Tab Icons


  Small Button Icon should have a height of 12px

  Medium Button Icon should have a height of 14px

  Large Button Icon should have a height of 16px


List Icons



            The flat edge of the collapse and expand caret should have a length of 9px

            The square collapse and expand icon should have a height of 14px
