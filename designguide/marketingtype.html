Print Typography


Typeface

The preferred typeface to use is Effra. If using Adobe Typekit to sync the Effra typeface is not an option, use Source Sans Pro (Google Font) or Helvetica.

  • Headlines should be set in Effra Light
  • Subheads should be Effra Medium
  • Body copy should use Effra Light
  • For emphasis copy use Effra Medium or Bold
