Color


Color Palette

Color is intended to be used as an accent. When in doubt, use color only for Headlines, Subheads, charts and infographics. The default background color should always be white. If color is used, be sure to follow the logo guidelines.

Remember to keep things light and airy. If your use of color makes the overall composition feel dark and claustrophobic, itâ€™s time to start removing elements to open things up.
