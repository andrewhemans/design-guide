Print Typography


Typeface

The preferred typeface to use is Effra. Use Adobe Typekit to sync the Effra typeface, if this is not an option, use Source Sans Pro (Google Font) or Helvetica.

  • Headlines should be set in Effra Light
  • Subheads should be Effra Medium
  • Body copy should use Effra Light
  • For emphasis copy use Effra Medium or Bold

Two Column Layout

When dealing with large amounts of text, a two column layout is preferred to one body column of text. Less is more, avoid unnecessary rules or solid blocks of colors. Layouts should feel light and airy; not dark and dense
